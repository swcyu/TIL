## L1 - New.
- 개인정보 수집 유효기간
``` python
def solution(today, terms, privacies):
    answer = []
    terms_dic, priv_dic = {}, {}
    for t in terms:
        terms_dic[t.split(' ')[0]] = int(t.split(' ')[1]) * 28
    ty, tm, td = today.split('.')
    today_days = int(ty)*12*28 + int(tm)*28 + int(td)
    
    for i, v in enumerate(privacies):
        py, pm, p = v.split('.')
        pd, pc = p.split(' ')
        calc_days = int(py)*12*28 + int(pm)*28 + int(pd) + terms_dic[pc]
        if calc_days <= today_days:
            answer.append(i+1)  
    
    return answer
```

- 주문량이 많은 아이스크림들 조회하기
```sql
SELECT FH.FLAVOR
FROM FIRST_HALF FH JOIN JULY J
ON FH.FLAVOR = J.FLAVOR
GROUP BY FH.FLAVOR
ORDER BY (SUM(FH.TOTAL_ORDER) + SUM(J.TOTAL_ORDER)) DESC
LIMIT 3;
```
